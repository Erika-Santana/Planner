# Atividade avaliativa da matéria de Dispositivos Móveis do curso de Análise e Desenvolvimento de Sistemas

### Descrição do projeto:
##### O aplicativo desenvolvido trata-se de um Planner, um tipo de agenda digital que busca auxiliar o usuário a ter controle das suas tarefas e responsabilidades.

### Funcionalidades:

#### Cadastro:
O usuário poderá realizar o seu cadastro, inserindo seu email, sua senha e caso deseje, uma imagem para seu perfil de preferência.

<img width="360" height="725" alt="image" src="https://github.com/user-attachments/assets/aeb5811f-614b-4753-a80a-376d84ad7b32" />

#### Login:
Com os dados cadastrados, o usuário poderá realizar o seu login:

<img width="357" height="719" alt="image" src="https://github.com/user-attachments/assets/a8d89935-983a-427b-9ffb-6bad5b4c5cfc" />

#### Home:
Ao entrar com o login, o usuário terá a visão das tarefas do dia, e também logo abaixo os botões para visualizar as tarefas registradas para os dias posteriores.

<img width="360" height="710" alt="image" src="https://github.com/user-attachments/assets/310a9b55-701d-4098-b844-c9072e0c9743" />

#### Alterar perfil:
Caso o usuário queira mudar os dados como nome e foto de perfil, ele poderá fazer isso na aba na barra de navegação...
<img width="362" height="744" alt="image" src="https://github.com/user-attachments/assets/d618193b-3dad-4efd-aed1-252fb0c09452" />

#### Criar Tarefa:
O usuário vai poder criar quantas tarefas quiser, adicionar imagens, colocar a descrição por transcrição da fala, e 
dessa forma receberá notificações de quando a tarefa estiver no horário que foi programado.
